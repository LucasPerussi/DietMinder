DietMinder is an app to manage your diet progress, with many features that might keep your goals on track.


Installing dependencies: 

```console
  npm install
```

Starting server in dev mode:

```console
  npm run dev
```

Starting prodution server:

```console
  npm start
```

## About:

```browser
  http://localhost:3000

```
